使用方法:

  1. 填写字段: 用户名, 密码, 验证码, profile名字

    *若已设置了认证器, 验证码可通过二维码提取获得. 二维码可通过认证器右上角-Export accounts类似的选项导出, 使用例如 https://github.com/scito/extract_otp_secret_keys 工具转换
    *请勿在公共空间内保留个人密码和验证码

  2. 如果你使用Xshell, 设置:
    工具[tools] - 选项[Options] - 常规[other]-Others-SSH “Always terminal based”

  3. 新建ssh登录会话:
    File - New - Connection 将协议[Protocol]设置为LOCAL
    登录脚本[Login Scripts]中选择”执行”, 并找到脚本位置添加, 例如”totp.py”
